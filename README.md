# walruslint

Walruslint for super quick eslint configuration!

### Install
`npm install walruslint -D`

### Setup
Go ahead and make yourself an .eslintrc and import walruslint.
```
{
  "extends": "walruslint"
}
```

### Peer Dependencies
`eslint`  
`babel-eslint`  
`eslint-config-airbnb`  
`eslint-plugin-import`  
`eslint-plugin-jsx-a11y`  
`eslint-plugin-react`
